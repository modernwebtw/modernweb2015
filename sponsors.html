白金級

Amazon Web Services (AWS)

Amazon Web Services 於2006年推出，開始將主要的IT基礎設施以網路服務的形式，公開讓企業使用 – 此形式也是目前大家耳熟能詳的雲端運算。

雲端運算與AWS的終極效益為其運用全新的商業模式，將IT重大基礎建設支出轉為可變成本。企業不再需要提早數週或數個月的時間，規劃與採購伺服器及其他IT資源。企業可藉由AWS，在需要IT資源時，借重Amazon的專業知識與規模經濟獲得這些資源，使企業能更快、並以更低的成本展現成果。

Amazon Web Services現今在雲端提供高可靠性、高擴充性，以及低成本的基礎架構平台。AWS雲端平台已是全球190個國家、數以萬計的企業、政府及新創事業客戶發展業務的重要推手。

Amazon Web Services 提供超過40種的不同服務，包括 Amazon Elastic Compute Cloud (Amazon EC2)、Amazon Simple Storage Service (Amazon S3) 及 Amazon Relational Database Service (Amazon RDS)。目前，全球客戶都可透過互聯網使用數據中心設置在美國、巴西、歐洲、日本、新加坡及澳洲等地的AWS雲端服務。

Nulab Inc.

Nulab是一間成立於 2004 年的日本敏捷式軟體開發公司，致力於開發協作軟體，藉以推動及促進個人和企業之間有效地溝通與協同合作。在台灣、東京、紐約、新加坡、福岡、和京都均設有辦事處以服務全球超過160萬用戶，目前的產品有：Cacoo - 即時線上協作繪圖工具，貝格樂 - Web版專案管理工具，Typetalk - 工作團隊即時通。

群傳媒股份有限公司(Readmoo電子書)

Readmoo電子書（以下簡稱Readmoo）為一結合書籍銷售、閱讀社群及創作市集的多元電子書服務平台。網站採用先進的 HTML 5 網頁技術，服務可跨平台、跨載具、離線閱讀，並強調畫線、註記、分享等社群功能。Readmoo目前已與超過 300 家華文出版社／作者合作，並積極與國際接軌，引進海外合作對象。

Readmoo母公司為群傳媒股份有限公司，總部設於台灣台北市。

英特威資訊股份有限公司

Interwave Information Corp.

英特威資訊專注於提供企業與組織關於WiFi、4G等行動網路解決方案。我們代理與銷售國際知名品牌網路通訊產品，並致力於銷售通路的支援。我們擁有相當的建置經驗，並參與相關政府、教育、企業等無線網路環境規劃與執行。

行動裝置爆炸性成長與臨時行動網路的市場需求，英特威資訊於國內首創無線網路活動行銷服務–WiFi Connect。提供有線和無線（WIFI/4G）的解決方案與活動行銷支援給貿易展覽、會議與企業活動。無論客戶的事件活動在室內還是室外，10人到10萬人與會者，我們有專業的團隊可以提供數個解決方案來快速滿足客戶的需求和預算。
